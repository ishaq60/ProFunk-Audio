# Frontend Task

This project implements the front-end design for "ProFunk Audio," an online library specializing in high-quality, royalty-free sound effects (SFX) and music tracks.

## Project Structure

```
ProFunk Audio/
│
├── index.html
│
├── assets/
│   ├── css/
│   │   ├── base.css
│   │   ├── layout.css
│   │   ├── components.css
│   │   ├── sections.css
│   │   └── utilities.css
│   │
│   ├── js/
│   │   ├── main.js
│   │   ├── utils.js
│   │   └── components.js
│   │
│   ├── images/
│   ├── icons/
│   └── fonts/
│
└── README.md
```

## Technologies Used

- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
